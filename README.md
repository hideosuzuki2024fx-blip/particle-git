# Particle Git Evolution System (v1.0)

This repository implements a pseudo-Git structure for tracking the evolution of knowledge particles generated by GPTs.

## Structure
- /particles/ : individual JSON knowledge particles
- /summary/   : CSV or aggregated data
- /meta/      : meta rules and schema definitions

## Principles
- Follow the principle of Non-Collaborative Rigor.
- Reliability Score < 0.9 cannot be merged into main.
- Unsupported assertions are recorded as draft (Score=0.0).
